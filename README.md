# 🚀 RescueLead

Sistema híbrido de prospecção ativa via Google Maps.

## 🛠️ Tecnologias
- **Node.js**: Motor de busca (Puppeteer).
- **PHP**: Inteligência de dados e Excel.
- **SQLite**: Banco de dados local.

## 📂 Como Executar
1. Configure o `.env` com nicho e cidade.
2. No terminal: `cd scripts-js && node engine.js`.
3. Os leads filtrados (nota < 4 e sem site) aparecerão no console.
